#    - Extract to C:\ghidra (Windows) or /opt/ghidra (Linux)
#    - Set GHIDRA_INSTALL_DIR environment variable

#    - Linux: sudo apt install gcc clang clang-format cppcheck
#    - macOS: xcode-select --install && brew install clang-format cppcheck
#    - Windows: Install MinGW-w64 or Visual Studio Build Tools

#    - clang-format: Code formatting

#    - black: Python code formatting (pip install black)
